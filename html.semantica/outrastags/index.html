blockquote marca uma citação extensa e recebe um atributo cite com a fonte.

Não devemos parar de explorar, e o fim de toda nossa exploração será chegar onde começamos e conhecer o lugar pela primeira vez

q marca uma citação inline (no texto).

Como já dizia o poeta se HTML está difícil, nem comece com JavaScript

cite é utilizada para citar o nome de um trabalho, livro, filme, peça ou algo parecido. Não é para citar autor.

Eu indico o livro Mythos escrito por Stephen Fry

em é utilizada para dar enfase em uma frase/palavra no texto.

Nenhum animal foi maltratado na produção deste conteúdo.

strong é utilizada para marcar uma parte importante do conteúdo

É importante entender como os frameworks funcionam. Porém é essencial aprender JavaScript.
