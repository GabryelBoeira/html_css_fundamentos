Este projeto tem dois pacotes, que se resumem em:

Projeto htmlComSemantica -> Nele possui tags separando as informações do código, o que, na prática, é uma boa prática de desenvolvimento web, pois os recursos de acessibilidade, neste caso os leitores de tela, interagem melhor com a aplicação desenvolvida.

Projeto htmlSemSemantica -> Nele o código possui o mesmo projeto e resultado do anterior, porém, sem as tags, os programas de acessibilidade não conseguem realizar, com a mesma qualidade e clareza, a leitura do conteúdo do site. Isso gera muita dificuldade para quem precisa utilizar esse recurso no sistema.
